* Commandline
#+BEGIN_SRC shell
  top
  top -b          # Batch-mode (not refreshing, just append the output)
  top -n <number> # Number of iterations. After refreshing number times, top exits.
  top -p <pid>    # Monitor PIDs
#+END_SRC
