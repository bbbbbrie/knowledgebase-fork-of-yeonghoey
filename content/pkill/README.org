* pkill
- Same as [[../pgrep][pgrep]], but kills matching processes
- Default singal is ~SIGTERM~
- ~kill -l~ to list signal names

#+BEGIN_SRC shell
  pkill -9 python     # You can also specify singal
  pkill -INT -f dynamodb
#+END_SRC
