* REST API v3
- https://developer.github.com/v3/

By default using ~https://api.github.com~, you request to v3 API.
To specify the version explicitly:
#+BEGIN_EXAMPLE
  Accept: application/vnd.github.v3+json
#+END_EXAMPLE
