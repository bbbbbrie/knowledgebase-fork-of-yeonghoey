** Conditional Replacement
#+BEGIN_EXAMPLE
  This text is always shown.

  {{#FieldName}}
  This text is only shown if FieldName has text in it
  {{/FieldName}}

  {{^FieldName}}
  This text is only shown if FieldName is empty
  {{/FieldName}}
#+END_EXAMPLE
